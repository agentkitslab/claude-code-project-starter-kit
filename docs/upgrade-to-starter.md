# Upgrade to the Starter Package

The free repo is intentionally small. It gives you a safe baseline.

The Starter package is for people who want a more complete Claude Code project operating system.

## Starter adds

| Area | Free | Starter |
| --- | --- | --- |
| `CLAUDE.md` | Yes | Yes |
| Rules | 5 core rules | 5 core rules |
| Commands | review, fix issue | review, fix issue, plan, docs update, release checklist |
| Agents | code reviewer, security auditor | + test designer, docs writer, release manager |
| Hooks | 2 examples | 2 examples |
| Skills | No | deploy checklist skill |
| Visual docs | No | structure diagram |
| Packaged ZIP | No | Yes |

## When to upgrade

Upgrade if you:

- Use Claude Code several times per week.
- Want repeatable release and documentation workflows.
- Need more specialized reviewer agents.
- Want a ready-to-copy paid package instead of assembling your own.
